# Contributing

This repository is the public Swift Package Manager distribution point for the Unloq Offers iOS SDK.

The SDK source is maintained internally by Unloq, so this repository does not accept general feature pull requests.

Useful contributions include:

- Documentation corrections
- Broken link reports
- Installation or versioning clarifications

For SDK behavior changes, integration support, or production rollout questions, contact the Unloq team through your partner support channel.
